# API Endpoints

Base URL: `http://localhost:8080/api/shopping`

- `GET /api/shopping` — list items (sorted by `updatedAt` desc)
- `GET /api/shopping/{id}` — get item by id
- `POST /api/shopping` — create item
- `PUT /api/shopping/{id}` — update item
- `DELETE /api/shopping/{id}` — delete item (204)
